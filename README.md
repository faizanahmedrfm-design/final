# SkinDuced Luxury Aesthetic Clinic Website

Vercel-ready Next.js 15 project.

## Upload to GitHub

1. Extract this ZIP.
2. Open the extracted folder.
3. Upload the files inside this folder to GitHub.
4. Make sure `package.json` is visible on the first page of the GitHub repo.
5. Import the repo in Vercel.
6. Keep Root Directory as `./` only when `package.json` is visible in the repo root.

## Vercel settings

Framework Preset: Next.js
Install Command: npm install
Build Command: npm run build
Output Directory: .next

## Local run

```bash
npm install
npm run dev
```

Open http://localhost:3000
